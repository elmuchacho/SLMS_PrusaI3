# Summary

<h2>This is YACS. (yet another calibration square)</h2>

<h3>Update 30-10-2017</h3>
<ul>
<li>Added vertical test piece</li>
</ul>

<h3>Update 26-10-2017</h3>
<ul>
<li>Flat corners now stand on the exact square corner.</li>
<li>Corners are now named.</li>
</ul>

The main difference from most other calibration tests is that the corners are clipped so that you can easily measure the X-Y skew of you printer. Nothing too fancy.

You can fix either try to fix it mechanically or use a gcode compensation tool.
GSuite is an example. You can downloaded in this [link].
 (http://pilotpage.monosock.org/fileadmin/files/Gsuite_v1.4.rar)
